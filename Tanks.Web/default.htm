Tanks

A Silverlight game by Mike Hodnick [www.kindohm.com]

"Tanks" is a game developed using Silverlight, C#, and the Farseer Physics engine.

Go and play the game.

Description

The game is a single-person game in which the object is to get as high of a score as possible. The player accumulates points by destroying enemy tanks, shooting efficiently, and minimizing hits taken from enemy tanks. The player can gain powerups to increase their rate of fire, hit points (health), or weapon types.
